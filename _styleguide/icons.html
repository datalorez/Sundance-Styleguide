Icons

We use the Material Design Icons. There are multiple ways to include an icon.

via SASS

To include an icon as a psuedo class (before or after).


@include icon(before, check) {
  /*
  Specific Styles
  */
}

via HTML

To include an icon as a psuedo class (before or after).


<i class="mdi mdi-check"></i>
